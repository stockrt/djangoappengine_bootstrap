h2. djangoappengine_bootstrap

Script to build a complete but simple test project running on
"djangoappengine":https://github.com/django-nonrel/djangoappengine / "django-nonrel":https://github.com/django-nonrel/django-nonrel with all deps bundled within.

Fully functional sample project created using this bootstrap script:
"https://github.com/stockrt/djangoadminongae":https://github.com/stockrt/djangoadminongae
"http://djangoadminongae.appspot.com/admin":http://djangoadminongae.appspot.com/admin

h2. Using

<pre>
<code>
    # clone
    git clone git://github.com/stockrt/djangoappengine_bootstrap.git
    cd djangoappengine_bootstrap

    # run
    chmod 755 djangoappengine_bootstrap.sh
    ./djangoappengine_bootstrap.sh

    # result
    Your test/sample app will be sitting at /tmp/djangoappengine_bootstrap
</code>
</pre>
